Projects

Some awesome Open Source Projects I have done in my journey of Data Science. Stay Tuned ! Some awesome project coming SOON!


Impact of Lockdown In India amidst COVID-19

Analysis and study of impact of Lockdown in India using various factors such as growth factor, mortality rate, testing rate, use the ARIMA model for forecasting and determination of recovery rate and contagion rate using SIR Model.



Analysis of Misinformation spread amidst COVID-19

In this project I have scraped the data from various IFCN certified fact checking website and categorize the misinformation into 8 different categories using LDA Algorithm and then analyze the type of misinformation spread with respect to different events, time period in India using NLPs, Word Cloud and Bar Plots.



Cancer Diagnosis Using Machine Learning

In this project I have used various Machine Learning algorithm like Logistic Regression, SVMs, KNNs, Random Forest and XGBoost to classify the cancer type using the gene feature and text literature feature.



Self Driving Car

The tensorflow implementation of Nvidia paper, in which I have build a Self Driving Car using end-to-end Convolution Neural Network(CNN) Algorithm, The data set I have used consists of images taken from car front dashboard along with steering angle, I divided the data set into train for training CNN model and after that testing the model.



Facebook Friend Recommendation Using Graph Mining

This project is based on social media link prediction whether two users are going to be friend in future or not. Generated training samples of good and bad links from given directed graph and for each link got some features like no of followers, is he followed back, page rank, katz score, adar index, some svd fetures of adj matrix, some weight features etc. and trained ml model based on these features to predict link.



Qoura Question Similarity Pair

The main task was to identify the duplicates questions asked on Quora. I focused on finding the number of unique questions, occurrences of each question, along with Feature Extraction, EDA and Text Preprocessing. I also explored Advanced Feature Extraction (NLP and Fuzzy Features) , Logistic Regression & Linear SVM with hyperparameter tuning. I also generated a WordCloud Generation.



Amazon Fine Food Reviews

In this project, I have done the semantic analysis of Amazon Reviews, by Preprocessing the text reviews using NLP techniques like BagofWorkd, Tfidif and Word2Vec. After that I used various machine learning algorithm like SVM, Random Forest and Xgboost.



Human Activity Recognition

This project is to build a model that predicts the human activities such as Walking, Walking Downstairs, Sitting and Laying from the dataset collected from sensors of smartphone.
